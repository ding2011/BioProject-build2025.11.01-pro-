# CHANGELOG
## [build2025.11.01] - 2025-11-01
### 新增功能
- 工业级多线程批量处理，支持20个文件并行分析
- 引物设计新增二级结构风险评估（发夹/二聚体检测）
- 可视化模块新增GC含量滑动窗口图、ORF分布图
- 数据库持久化：分析历史、序列缓存自动存储（30天过期）
- 内存监控与自动清理功能，优化长序列处理稳定性

### 优化改进
- 修复Tm值计算异常问题（新增Wallace规则回退机制）
- 优化ORF预测算法，支持真核生物内含子剪接位点检测
- 界面适配高分辨率屏幕，新增语法高亮（ATCGU碱基彩色显示）
- 错误恢复机制：MAFFT比对失败自动重试（最多3次）

### 已知限制
- 多序列比对仅支持≤50条序列（长序列建议≤20条）
- 进化树构建目前仅支持NJ法，后续将新增ML法
